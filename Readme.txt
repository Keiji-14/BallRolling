【制作使用エンジン】Unity

【制作期間】一週間

【バージョン管理】SourceTree

【ゲームジャンル】ボール転がしゲーム

【ゲーム概要】
・このゲームはボールを転がしてゴールさせるゲームです。

・タイトル画面からステージ選択画面で遊ぶステージを選択します。

・ゲームではボールをステージの奥にあるゴールの穴に入るとステージクリアとなります。

・ステージにて壁はなく、ステージの場外に出ますと落ちてしまいゲームオーバーとなります。

・ステージは６種類用意しております。

【操作説明】
タイトル画面：
・画面上に表示しているボタンをクリックで選択します。

ゲーム画面：
・WASDキーまたは↑↓→←キーでプレイヤー転がしたい方向にキーを押します

【アピールポイント・苦労した点】
・一週間という短期間で一つのゲームを遊べるように制作しました。

・ゴールを Unityの「ProBuilder」ツールを使用してモデリングしました。

・Unityで「EditorWindow」を作成し、Sceneビュー上をクリックする事でオブジェクトを生成・配置するツールを開発しました。

・ツールを作成する際に、配置したオブジェクトが重なったりしないようにグリッド線を引き、グリッド線を基準にオブジェクトを配置するように工夫しました。
